# IPL-2022-Data-Analysis-Project

## 📌 Overview
This project analyzes IPL 2022 match data to uncover insights about team performance, player statistics, and match outcomes.

## 📊 Key Insights
- Gujarat Titans won the most matches
- Toss decisions impacted match results
- Analysis of wins by runs and wickets
- Top players like Jos Buttler and Kuldeep Yadav performed consistently
- Highest scores and best bowling performances identified
- Venue-based match analysis

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## 📈 Learning Outcomes
- Data Cleaning
- Data Analysis
- Data Visualization
